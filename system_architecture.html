Technology Stack

Frontend: React.js, HTML5, CSS3, JavaScript

Backend: Node.js, Express.js, JWT Authentication

Machine Learning: Python, scikit-learn, pandas, numpy

Data Storage: JSON files, Pickle models

APIs: RESTful services, Content-based recommendations

Key System Components

Authentication System: JWT-based user authentication and authorization

Recommendation Engine: Hybrid ML approach with content-based and collaborative filtering

Symptom Analysis: Multi-symptom processing with specialist mapping

Rating System: User feedback collection and integration

Real-time Updates: Dynamic doctor availability and scheduling
